Design System Foundation

Every element on this page follows the same rules. This is the structural scaffold — no product features, only the system.

Color Tokens

Background #F7F6F3, Text #111111, Accent #8B0000, plus muted green and amber for states. Maximum four colors across the entire system.

Typography Scale

Serif headings at generous sizes with tight leading. Sans-serif body at 16–18px with 1.6–1.8 line-height. All text blocks capped at 720px width.

Spacing System

Five values only: 8px, 16px, 24px, 40px, 64px. No exceptions. Whitespace is part of design, not an afterthought.

Component Rules

Same border-radius, same hover timing, same focus ring everywhere. Cards use subtle borders, no drop shadows. Inputs use clean borders with clear focus state.
